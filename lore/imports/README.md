# Lore Imports

Dieser Ordner enthaelt perspektivische Importdateien gemaess `lore/lore_import_schema.md`.

Namensschema:
- `import_<YYYY-MM-DD>_<slug>.md`

Ablauf pro neuem Import:
1. Quelle unveraendert in `lore/raw/` sichern
2. Importdatei in diesem Ordner aus `import_template.md` erstellen
3. Ely-, Lauri-, George- und Spielleiter-Sicht getrennt erfassen
4. Sichtbarkeit und Relevanz fuellen (OFFEN/VERDECKT/HINTERGRUND, Trigger, Relevanz jetzt/spaeter)
5. Konsolidieren und in Strukturdateien mappen
6. Tonanker und Zitate in `lore/lore_flair_quotes.md` uebernehmen
