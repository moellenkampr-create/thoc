# thoc

Dieses Repository organisiert THoC in mehrere Chat-Instanzen mit eigenen Startdateien und Ordnerstrukturen.

## Startdateien
- `manifest/manifest_thoc.md` – globales Manifest
- `meta/start_meta.md` – Meta/Advisor Startdatei
- `lore/start_lore.md` – Lore Startdatei
- `foundrycwn/start_foundrycwn.md` – FoundryCWN Startdatei
- `foundryurop/start_foundryurop.md` – FoundryURoP Startdatei
- `manga/start_manga.md` – Manga Startdatei
- `band/start_band.md` – Band Startdatei
- `urop/start_urop.md` – URoP Startdatei

> Hinweis: Neue Startdateien und relevante Chat-Dateien werden hier ergänzt, sobald sie ins Repository aufgenommen werden.
