# URoP Governance
Stand: 2026-04-11

## Zweck
Dieses Dokument steuert den Arbeitsmodus fuer Folgechats und Merge-Paesse.

## Rollenmodell
- Regeln und Balancing: URoP-Entwicklung
- Lore und Kanon: externe Weltentscheidung

## Verbindliche Prinzipien
- Basissystem schlank halten
- Komplexitaet modular andocken
- JSON fuehrt Datenbloecke
- Regeltexte bleiben fiction-first lesbar
- Keine stillen Regelwechsel ohne Nachziehen in Index und Backlog

## Statusmarker
- GELOCKT
- TESTSTAND
- ARBEITSSTAND
- OFFEN

## Pflegepflicht bei Aenderungen
1. Betroffene Regelquelle aendern
2. Falls Datenblock: zuerst data, danach ggf. Lesefassung
3. docs/governance/BACKLOG.md aktualisieren
4. docs/governance/INDEX.md aktualisieren, wenn Dateien hinzukommen/entfallen
5. Bei Architekturfragen ADR in docs/decisions erstellen oder aktualisieren

## Exportregel
Triggerbegriffe: SYNC_EXPORT, EXPORT, CHAT_EXPORT, FOLGECHAT_EXPORT

Ablauf:
1. Gesamtstand aktualisieren
2. Konsistenz pruefen
3. ZIP erstellen
4. ZIP und Kernquellen bereitstellen
