# URoP – Charaktererschaffung
Stand: 2026-04-08

## 1. Grundsatz [GELOCKT]
Charaktere werden nicht nur als freier Ein-Topf-Build gedacht, sondern **stufenweise über Lebensphasen**.

---

## 2. Ausrichtung [GELOCKT / TESTSTAND]
Die Ausrichtung ist die erste große Charakterentscheidung.

### 2.1 Mögliche Ausrichtungen [GELOCKT]
- Körper
- Geist
- Präsenz
- Körper/Geist
- Körper/Präsenz
- Geist/Präsenz
- Ausgewogen

### 2.2 Leitlinie [GELOCKT]
Ausrichtung soll **formen, nicht verbieten**.

### 2.3 Kostenmodifikationen [TESTSTAND]
Ausrichtung wirkt aktuell auf die **Grundkosten** von Fertigkeiten, nicht auf Überhang.

#### Mono-Ausrichtung
- passender Bereich: -20 %
- beide anderen: +10 % / +10 %

#### Mischausrichtung
- zwei passende Bereiche: -10 % / -10 %
- dritter Bereich: +20 %

#### Ausgewogen
- keine Modifikation

Rundung zugunsten des Spielers.

---

## 3. Spezialattribute / Sondernatur [GELOCKT]
Spezialattribute sind optional.
Sie sind kein Pflichtteil des Kernsystems.

Leitlinie:
Spezialkräfte sind **nicht fest an Geist** gebunden, sondern können an Körper, Geist oder Präsenz andocken.

---

## 4. Lebensphasen [GELOCKT]
### Reihenfolge
1. Ausrichtung
2. Sondernatur / Spezialattribut
3. Kindheit / Herkunft
4. Jugend / frühe Prägung
5. Ausbildung / Formung
6. Frühe Erwachsenenerfahrung
7. Aktuelle Position / Startstatus

### Leitlinie
Lebensphasen geben **Schwerpunkte**, keine harten Verbotszonen.

### Spezialisierung in Lebensphasen [GELOCKT]
Spezialisierungen können in **jeder Lebensphase** entstehen.

---

## 5. Verfahrensregel pro Phase [ARBEITSSTAND]
1. Leitfragen klären
2. Relevanz bestimmen
3. typische Folgen ableiten
4. mechanisch eintragen
5. in die nächste Phase überführen

---

## 6. Budgetmodell [TESTSTAND]
Aktuelle Richtung:
- **Gesamtbudget + empfohlene Phasenverteilung**
- kein reines Paketmodell

### Aktueller Testwert
- **600 Entwicklungspunkte**

### Empfohlene Phasenanteile
- Kindheit / Herkunft: 10–15 %
- Jugend / frühe Prägung: 15–20 %
- Ausbildung / Formung: 25–30 %
- Frühe Erwachsenenerfahrung: 25–30 %
- Aktuelle Position / Startstatus: 5–10 %

---

## 7. Attributkosten [TESTSTAND]
Ausgehend von Basis 2:
- 3 = 40
- 4 = 90
- 5 = 160
- 6 = 260

---

## 8. Spielbarer SC bei Start [ARBEITSSTAND]
Ein Start-SC sollte in irgendeiner Form vier Bereiche abdecken:
1. Kernkompetenz
2. Selbstbehauptung / Überlebensfähigkeit
3. Alltags- und Handlungstauglichkeit
4. Anbindung an Gruppe, Welt und Startlage

Das ist kein starres Verbotsschema, sondern ein Prüfraster.
